# 🏏 Hardik Pandya Cricket Performance Dashboard

## 📌 Project Overview

This project presents an interactive **Power BI Dashboard** analyzing Hardik Pandya's batting performance across:

- Test Cricket
- One Day Internationals (ODI)
- T20 Internationals (T20I)

The dashboard provides insights into:

- Total Matches
- Total Runs
- Highest Score
- Total 50s
- Year-wise Performance Trends
- Format-wise Comparison

---

## 🎯 Objectives

- Analyze Hardik Pandya's batting statistics.
- Compare performance across cricket formats.
- Identify yearly trends in runs scored.
- Visualize milestones such as half-centuries.
- Create an engaging and interactive sports analytics dashboard.

# 📊 Insights

### ODI
- Highest contribution of runs among all formats.
- Strong performance in 2017 and 2023.

### T20I
- Significant improvement after 2021.
- Highest yearly runs recorded in 2022.

### Test
- Limited appearances but impactful performances.
- Recorded a century with a highest score of 108.
